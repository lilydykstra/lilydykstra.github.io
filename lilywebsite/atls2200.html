ATLAS 2200: Web

These are all websites I have made in my Web class. We started the class with basic HTML, then moved onto CSS in order to style our websites, and finally added JavaScript to add interactions. They will all direct you to a new page not in this website.

In Class Activities

Birdwatching Website: For this project, our task was to take the content for the homepage of a bird watching website and add structural elements to it so it can have a page layout applied to it as an introduction to HTML

Intro to CSS: We had to adjust a given file to a pst of css attributes to make this the end result.

Advanced CSS: We were given raw html and css and required to format it in a certain way, now shown.

CSS Layouts Pt.1: We needed to position everything as told on the screen.

CSS Layouts Pt.2: We had to design the elements as described on the screen.

CSS Layouts Pt.3: We had to use media queries to make all the conditions true.

CSS Animation and Transitions: We had to make the boxes do what they say: change color at hover and move around.

Interactive Elements: We created event psteners for the button to create the designs each time it is clicked.

Trivia Machine: We connected an API to our website to get christmas trivia and the answers at the click of the buttons.

Weekly Assignments

Encyclodeia on Rhinos: We created an encyclopedia page on our website. We picked something that we know well (or want to learn more about) and made a brief encyclopedia page.

Business Card: We were provided with some raw HTML and an image, and we assigned to write the necessary CSS to style it into an onpne business card of a certain design.

Overlays: We were tasked with making two overlays on top of a base image.

CSS Layouts: We were given a website and had to organize it to look this way using flexbox and grid elements

Animate Something: Our assignment was to edit the starting files and add an animation to the website.

Silly Story Generator: We had to add all the interactions and adjust the content and design of the generator.

Image Carousel: We created click interactions to change the big image on display and the thumbnail images, as well as the darken/pghten button.

API Generator:We had to connect an external API and make our own "quote" generator when the button is pressed.

Midterms/Finals

HTML Midterm: Our HTML Midterm required us to fix all the errors on each page of the website and additionally create a page of our own where we log the errors we fixed.

CSS Midterm: Out CSS Midterm required us to take a bunch of information about the Colorado Symphony and organize it on a website in a pleasing and self explanatory way.
